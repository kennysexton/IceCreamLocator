Media queries

Media queries are a way of delivering different CSS based on factors. For our navbar we will use media queries that check the size of the users view port. When the user is viewing the page on a screen size that is smaller then the media query, then different CSS rules are applied.

Hiding Elements

Combining media queries with the CSS display attribute. We can hide html elements depending on screen size. This is useful since it allows us to maximize code reuse. Instead of writing the HTML code for two different menu's, we can create one menu and use CSS to hide elements that appear on the mobile menu.

Transition

We can combine hiding an element with a transition to create a sidebar that transitions on appearance. 


            function openNav() {
                document.getElementById("mySidebar").style.width = "250px";
              }
            function closeNav() {
                document.getElementById("mySidebar").style.width = "0px";
            };
            

            .sidebar {
                    height: 100%;
                    width: 0px;
                    position: fixed;
                    z-index: 10;
                    top: 0;
                    right: 0;
                    background-color: #111;
                    overflow-x: hidden;
                    transition: 0.5s;
                    padding-top: 55px;
                }
